无法正常使用GitHub的解决方法

发表于 2021-02-05 更新于 2021-08-05 分类于 瞎折腾 Valine：

由于众所周知的某些原因，GitHub（同性恋交友击剑网站）总是玄学的被墙，尤其是图片，很多时候我连自己的头像都看不到。
本片内容讲一讲如何在不使用魔法的情况下自由的击剑（雾）

这个问题的原因是GitHub域名被污染所以，我们可以更改本机hosts文件进行自解析来防止这些问题（但并不能解决下载速度慢这个问题，毕竟水管细，如果想快可以是用国内gitee做中转，或者使用魔法）

注意：该解析是在2月10日更新的，不过GitHub更换ip的频率很低，如果不行可以评论我会再次更新，也可以通过github社区的这个项目直接获得

使用方法

将上述代码框内的这些东西复制添加到你hosts文件后

*打开hosts文件** hosts文件路径C:\Windows\System32\drivers\etc

可以通过任意一款文本编辑器打开进行修改，我这里使用的是Sublime Text 3打开，也可以使用windows自带的记事本打开。让后复制我代码框中的东西，粘贴在所有文本下面

更改hosts后，注意在cmd中使用ipconfig /flushdns刷新DNS缓存

1.点击win键＋R键打开运行 2.输入cmd回车打开CMD

**在cmd的界面里输入ipconfig /flushdns 后回车 ** 来清除缓存

让后我们就可以快乐的浏览世界上最大的同性恋交友网站Github了！
